# Personal Portfolio Website

A responsive and interactive portfolio website developed using **React.js** to showcase my professional profile, technical skills, projects, certifications, education, and contact information. The website features a clean modern UI, smooth animations, and an organized navigation experience.

## Features
- Responsive design for desktop and mobile devices
- Modern React-based user interface
- Multi-page navigation using React Router
- About, Education, Projects, Certificates, and Contact sections
- Smooth animations using Framer Motion
- Firebase integration
- Downloadable resume
- Professional project showcase
- Easy-to-maintain component-based architecture

## Tech Stack
- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- React Router
- Firebase
- Framer Motion
- Lucide React Icons

## Project Structure
- Home
- About
- Education
- Projects
- Certificates
- Contact
